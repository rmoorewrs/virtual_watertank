# Watertank Controller written in C language

This is version of the virtual watertank controller written in C. 

Prerequisites are:
- libcjson -- used for JSON handling
- libcurl -- used to make API requests

To install on a Debian-based system:
```
sudo apt install -y libcjson-dev libcurl4-gnutls-dev
```
